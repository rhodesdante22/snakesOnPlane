Please keep all .m files within the provided folder. 

Main executable:
finalScript.m

Helper functions:


source_sink.m | Main contour generation method. 


paramOptimization.m | Optimizing parameters to use for
LossFn.m | Calculating loss for optimizing parameters 
getTruthContours.m | used for optimizing algorithm
getTable.m | Getting values from provided tables
getLen.m | helper function for xInterp
findNthPointGivenLast.m | helper function for xInterp
findLongAxisGivenBottom.m | Finding the long axis using found bottom point
findLongAxis.m | Finding the general long axis
findExtrema.m | finds min and maxes for ESV/EDV calculations
find_points.m | Finding initial reference points
find_area.m | Calculating area found within contour
ejectionFraction.m | computes ejection fraction
dodgeVolume.m | computes dodge volume estimate
determineTissueIndices.m | determines contour indices for tissue tracking
addEps.m | adds a small value to an array
xInterp | general purpose resampling function




Additional material 
*.mat | used for optimizing the algorithm
